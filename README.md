# Bangla-Image-Captioning
The goal of this project is to generate a descriptive Bengali (Bangla) caption for any given image. Taking any image as input the model outputs a sequence of Bengali words that describe the scenario in that image.

The full dataset used for training the model can be found here: https://data.mendeley.com/datasets/rxxch9vw59/2
The dataset was hosted and fetched from google drive during the course of training.
